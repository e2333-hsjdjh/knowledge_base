## 📚 Markdown 基础语法速查

为了方便编写文档，这里列出了一些常用的 Markdown 语法规则供参考：

### 1. 标题
使用 `#` 号表示标题，`#` 的数量对应标题级别。
```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 2. 文本样式
- **粗体**: `**文本**` 或 `__文本__`
- *斜体*: `*文本*` 或 `_文本_`
- ~~删除线~~: `~~文本~~`
- `行内代码`: 使用反引号 \` 包裹

### 3. 列表
**无序列表** (使用 `-`, `+`, 或 `*`):
- 项目 A
- 项目 B

**有序列表** (使用数字加点):
1. 第一步
2. 第二步

### 4. 引用
使用 `>` 表示引用：
> 这是一段引用文本，通常用于摘录或强调。

### 5. 代码块
使用三个反引号 \`\`\` 包裹代码，并指定语言（如 python, java, bash）：
\`\`\`python
def hello():
    print("Hello, World!")
\`\`\`

### 6. 链接与图片
- **链接**: `[链接文字](https://example.com)`
- **图片**: `![图片描述](path/to/image.png)`

更多高级语法可参考 [Markdown 官方教程](https://markdown.com.cn/)。
