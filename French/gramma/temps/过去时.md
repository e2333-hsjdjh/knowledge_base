# 2. 复合过去时 Le passé composé
`avoir + fait(le participe passé)` 或 `être + fait(le participe passé)`  
## 规则动词
- 以-er结尾的动词，过去分词以-é结尾，如：parler → parlé
- 以-ir结尾的动词，过去分词以-i结尾，如：finir → fini
- 以-re结尾的动词，过去分词以-u结尾，如：vendre → vendu
- 以-oir结尾的动词，过去分词以-u结尾，如：voir → vu
- 以-ire结尾的动词，过去分词以-it结尾，如：dire → dit
- 所有的一般词尾都是i,is,it,u
